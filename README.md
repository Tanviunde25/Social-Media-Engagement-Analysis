- Project Title

Social Media Engagement Dashboard

- Goal

Analyze likes, comments, and shares to understand audience engagement trends and content performance.

- Problem Statement

Social media platforms generate large engagement data. This project analyzes engagement metrics to help decide best content strategies and posting patterns.

- Dataset

1. File: social_media_engagement1.csv

2. Columns used:

* likes

* comments

* shares

- Technologies Used

1. Python

2. Pandas

3. Matplotlib

- Steps Performed

1. Load CSV file

2. Calculate average likes, comments, shares

3. Plot line graph for engagement metrics

4. Interpret engagement trends

- How to Run

pip install pandas matplotlib


Then:

python main.py


(or run main.ipynb)

Ensure social_media_engagement1.csv is in the same folder.

- Output

* Mean engagement values

* Line graph showing:

Likes

Comments

Shares

- Business Insights

* Identify best performing content

* Understand audience interaction

* Improve posting strategy
